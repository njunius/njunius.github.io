How to join anime night by Melanie:

If you don't mind lower quality audio, just ask if someone can stream it on Discord! No setup needed on your part!

If you want better quality:

  1. Download mpv (we use mpv b/c other video players don't handle special subtitles well; e.g., will sometimes crash with lots of fancy subtitles, which we often have with the cool fan subs Nick gets). Be sure to grab the latest stable version of mpv (v0.32.0, from Jan 26):

    • Windows: https://sourceforge.net/projects/mpv-player-windows/files/64bit/mpv-x86_64-20200126-git-ae0520c.7z/download
    • Mac: https://laboratory.stolendata.net/~djinn/mpv_osx/mpv-latest.tar.gz

  2. Download SyncPlay - the latest version should be fine. https://syncplay.pl/download/
    Server address: syncplay.pl:8997 (use the dropdown to select)
    Default room: EISTrolls
    no password
    Set the 'Path to Media Player' to wherever your mpv install lives

  3. Download whatever media files we're watching, usually posted in the anime-night channel on the Discord server a day or two before

  4. Once you're in the SyncPlay room, add the downloaded video files using File > Set media directories (so you can add a whole folder, instead of opening them one at a time)

FYI, mpv mainly uses keyboard controls, and doesn't really have a GUI, especially on Mac :P. Here's a list of key bindings:
https://mpv.io/manual/master/ (e.g., I'm always using 0/9 for volume control)

Also FYI, here's a spreadsheet for brainstorming next animes to watch, tracking watching progress, etc:
